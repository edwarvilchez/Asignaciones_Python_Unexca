# Asignaciones Introducción a la Programación con Python - UNEXCA

Este repositorio contiene las soluciones detalladas y el material de apoyo para las asignaciones de la cátedra de Programación en la **Universidad Nacional Experimental de la Gran Caracas (UNEXCA)**.

## 🎯 Objetivo de la Actividad
El propósito de esta asignación es que los estudiantes (mis "padawans") dominen los fundamentos básicos de Python, incluyendo:
* Manejo de variables y tipos de datos.
* Estructuras de control de flujo (If, For, While).
* Definición y uso de funciones.
* Lógica de resolución de problemas aplicada.

## 📺 Explicación en Video (Solucionario)
Para facilitar el aprendizaje, he grabado una serie de videos donde explico paso a paso la lógica detrás de cada solución. Puedes ver la lista de reproducción completa aquí:

👉 [**Lista de Reproducción: Solucionario Asignación 1 - Python**](https://www.youtube.com/playlist?list=PLonQCfhtJ5LvEfjZRd0_bg1VrFPUlo6Tm)

## 📁 Contenido del Repositorio
* `/src`: Contiene los scripts de Python con las soluciones a los enunciados.
* `Enunciado.md`: Descripción detallada de los problemas planteados en clase.
* `requirements.txt`: Dependencias necesarias (en caso de aplicar).

## 🚀 Cómo utilizar este material
1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/edwarvilchez/Asignacion_1_Python_Unexca.git](https://github.com/edwarvilchez/Asignacion_1_Python_Unexca.git)
    ```
2.  **Intenta resolverlo por tu cuenta:** Te recomiendo leer el enunciado e intentar codificar la solución antes de mirar el código resuelto.
3.  **Consulta el código y el video:** Si te quedas atascado, revisa el script correspondiente y mira el video explicativo para entender el "porqué" de cada línea de código.
